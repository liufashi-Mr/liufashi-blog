## 使用指南

### 通过脚手架

[npm地址](https://www.npmjs.com/package/react-client-create)

```shell
# global install
npm i react-client-create -g
# and then run
create-cli create [name]

# or
npm i react-client-create
#then run
npx create-cli create [name]
```
